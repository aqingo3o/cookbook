# 美麗的 keyword 顏色
配色靈感尖尖來自 python IDLE，根本是白月光等級的。  
我將會在所有電腦上使用這個配置直到有更美麗的顏色出現並說服我 ;)  

貼到 `setting.json`即可獲得美麗集成環境。
```
"workbench.colorTheme": "Default Dark+",
    "editor.tokenColorCustomizations":{
        "comments": "#ff5858",
        "functions": "#dba0db",
        "keywords": "#fcc642",
        "strings": "#a5cf27",
        "variables": "#ffffff",
        "numbers": "#cee6fe"
    },
    "workbench.colorCustomizations": {
        "editorBracketHighlight.foreground1": "#ffffff",
        "editorBracketHighlight.foreground2": "#ff7db5",
        "editorBracketHighlight.foreground3": "#7ac8ff",
        "editorBracketHighlight.foreground4": "#faff77",
    },
```
