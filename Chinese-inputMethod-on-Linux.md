# 裝中文輸入法
好我到大概有 200 個教程是在做這件事的，但就是紀錄一下成功的經驗

## 裝這個
聽起來很有嚼勁的東西
```
sudo apt install ibus-chewing
reboot
```

## 重開機之後
Setting -> Region & Language -> Input Sources -> 按下面的**加號**  
按下面的三個垂直排列的點點  
直接搜尋 "chewing"  
選擇 Chinese 那什麼的，按 **Add**  
這樣就有了

## 切換輸入法的方法
settings -> keyboard shortcuts -> "Typing" (一個大塊)  
switch to next input source，然後可以自訂快捷鍵
